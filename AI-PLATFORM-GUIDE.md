# 重要说明：AI搜索的两种场景

## 📌 当前系统实现的是什么？

### ✅ 已实现：自有AI搜索系统

**场景描述：**
```
用户访问您的网站 → 在您的网站上搜索 → 您的后端调用AI API → 返回企业推荐
```

**使用流程：**
1. 企业主在后台添加企业信息
2. 客户访问您的网站（如 http://your-domain.com/ai-search）
3. 客户在您的网站上搜索关键词
4. 您的系统调用AI API，返回匹配的企业信息

**这是一种"自建AI搜索服务"，类似于：**
- 美团商家后台
- 大众点评商家系统
- 小红书商家管理

---

## ❓ 您可能想要的是什么？

### 🎯 场景B：被豆包、文心一言等AI平台收录

**场景描述：**
```
用户打开豆包APP → 在豆包中搜索"附近川菜餐厅" → 豆包直接显示您客户的企业信息
```

**这需要：**
1. 将企业数据提交到豆包、百度、阿里等平台的**知识库**
2. 通过这些平台的**企业认证**和**数据接入**
3. 满足平台的**收录标准**和**审核要求**

---

## 🔄 两种场景对比

| 特性 | 场景A：自有AI搜索（当前） | 场景B：AI平台收录 |
|------|------------------------|-----------------|
| 用户入口 | 您的网站 | 豆包/文心一言等APP |
| 数据控制 | 完全自主 | 需要平台审核 |
| 收录条件 | 无限制 | 需要营业执照等 |
| 成本 | 低 | 可能需要付费 |
| 流量来源 | 您自己引流 | 平台自然流量 |
| 实现难度 | ✅ 已完成 | ⚠️ 需要对接 |

## 💡 解决方案：如何让企业被AI平台收录？

### 方案一：百度企业百科（推荐）

**步骤：**
1. 访问百度企业百科
2. 提交企业营业执照、法人信息
3. 创建企业百科词条
4. AI搜索时会引用企业信息

**优势：**
- 百度文心一言会优先引用
- 免费
- 权威性高

**链接：** https://baike.baidu.com/business/

---

### 方案二：高德地图/百度地图标注

**步骤：**
1. 在高德地图/百度地图添加企业地点
2. 提交营业执照、门店照片
3. 审核通过后，AI搜索会关联位置信息

**优势：**
- AI搜索"附近XX"时会展示
- 增加企业曝光
- 免费

**链接：**
- 高德：https://lbs.amap.com/
- 百度：https://lbsyun.baidu.com/

---

### 方案三：企业官网SEO优化

**步骤：**
1. 建立企业官网
2. 添加结构化数据（Schema.org）
3. 优化SEO
4. AI爬虫会自动抓取

**优势：**
- 所有AI搜索引擎都能找到
- 长期有效
- 成本可控

**技术要点：**
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Restaurant",
  "name": "美味轩餐厅",
  "servesCuisine": "川菜",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "北京市朝阳区"
  }
}
</script>
```
