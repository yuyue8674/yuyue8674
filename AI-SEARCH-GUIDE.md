# GEO获客系统 - AI智能搜索功能说明

## 🎯 核心功能升级

系统现已接入**真实AI搜索API**，实现了企业信息在AI智能搜索中的精准展示！

## ✨ AI智能搜索特性

### 1. 智能关键词匹配
- ✅ **精确匹配**：关键词完全一致，最高优先级
- ✅ **部分匹配**：关键词包含搜索词，中等优先级
- ✅ **广泛匹配**：语义相关匹配，低优先级

### 2. 地理位置优化
- ✅ 基于用户位置的智能推荐
- ✅ 地理围栏范围自动过滤
- ✅ 距离计算和优先级排序

### 3. AI驱动的交互体验
- ✅ 流式输出，打字机效果
- ✅ 友好的对话式回复
- ✅ 专业的企业推荐理由
- ✅ Emoji增强可读性

## 🚀 使用指南

### 步骤1：添加企业信息
1. 进入"企业管理"页面
2. 点击"添加企业"
3. 填写企业详细信息：
   - 企业名称
   - 业务类型
   - 联系方式
   - 地址和经纬度

### 步骤2：配置关键词
1. 进入"关键词管理"页面
2. 为企业添加搜索关键词
3. 设置优先级（数字越大越优先）
4. 选择匹配类型（精确/部分/广泛）

### 步骤3：设置地理围栏（可选）
1. 进入"地理围栏"页面
2. 为企业创建覆盖区域
3. 设置覆盖半径（公里）
4. 启用围栏

### 步骤4：测试AI搜索
1. 点击侧边栏"AI智能搜索"（带NEW标签）
2. 输入搜索关键词，如：
   - "附近好吃的川菜餐厅"
   - "咖啡厅"
   - "健身房"
3. 可选择输入位置信息
4. 点击"AI搜索"按钮
5. 查看AI实时流式输出

## 💡 最佳实践

### 关键词配置建议
1. **核心关键词**（优先级10）
   - 行业通用词：餐厅、咖啡、健身
   - 品牌特色词：川菜、粤菜、精品咖啡

2. **长尾关键词**（优先级7-9）
   - 位置词：附近餐厅、朝阳区咖啡厅
   - 服务词：24小时健身房、外卖餐厅

3. **场景关键词**（优先级5-6）
   - 需求词：减肥健身、商务聚餐
   - 氛围词：安静、浪漫、亲子

### 地理围栏设置建议
- 市中心商圈：半径5-10公里
- 社区周边：半径3-5公里
- 特定区域：半径2-3公里

## 📊 系统架构

```
用户输入关键词
    ↓
AI搜索API (/api/ai-search)
    ↓
从文件存储加载企业数据
    ↓
构建企业知识库
    ↓
调用LLM API（豆包大模型）
    ↓
流式输出AI推荐结果
    ↓
前端实时渲染
```

## 🔧 技术栈

- **AI引擎**：豆包大模型 (doubao-seed-2-0-pro-260215)
- **前端**：Next.js 16 + React 19
- **数据存储**：JSON文件持久化
- **UI组件**：shadcn/ui + Tailwind CSS
- **流式传输**：Server-Sent Events (SSE)

## ⚠️ 注意事项

1. **数据持久化**：所有数据保存在 `data/geo-data.json` 文件中
2. **API配额**：AI搜索使用豆包大模型API，有调用次数限制
3. **响应时间**：首次搜索可能需要3-5秒，后续会更快
4. **位置精度**：建议提供准确的经纬度以获得最佳推荐
