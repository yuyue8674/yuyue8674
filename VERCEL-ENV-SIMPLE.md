# 📱 Vercel环境变量配置（简化版）

---

## 🎯 环境变量配置

在Vercel的 **"Environment Variables"** 区域，您只需要：

### 操作步骤

**1. 找到输入框**

找到两个输入框：
- 第一个输入框（Key）
- 第二个输入框（Value）

**2. 输入内容**

在第一个输入框输入：
```
ARK_API_KEY
```

在第二个输入框输入：
```
24345fee-03c5-4013-b98a-28f889f157c2
```

**3. 点击添加**

点击 **"Add"** 按钮（或类似按钮）

---

## 📋 简化示意

```
┌─────────────────────────────────┐
│ Environment Variables           │
├─────────────────────────────────┤
│                                 │
│ [ARK_API_KEY    ]  ← 第一个框  │
│ [24345fee-03c5...]  ← 第二个框 │
│                                 │
│ [Add]  ← 点击这个按钮           │
│                                 │
└─────────────────────────────────┘
```

---

## ✅ 确认添加成功

添加成功后，您会看到：

```
ARK_API_KEY = 24345fee-03c5-4013-b98a-28f889f157c2
```

显示在环境变量列表中。

---

## 🚀 然后继续

添加环境变量后：

1. 滚动到页面底部
2. 点击 **"Deploy"** 按钮
3. 等待部署完成
