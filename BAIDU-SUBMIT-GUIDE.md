# 📱 百度站长平台添加网站详细步骤

---

## 🎯 目标

在百度站长平台添加您的网站：`https://yuyue8674.vercel.app`

---

## 📋 详细步骤

### 第1步：访问百度站长平台

在浏览器中打开：
```
https://ziyuan.baidu.com
```

---

### 第2步：登录账号

**方式1**：使用百度账号登录
**方式2**：使用手机号注册新账号

---

### 第3步：添加网站

**3.1 找到添加入口**

登录后，在页面中找到：
- **"添加网站"** 按钮
- 或 **"用户中心" → "站点管理" → "添加网站"**

**3.2 输入网站地址**

在输入框中输入：
```
https://yuyue8674.vercel.app
```

**3.3 点击提交**

点击 **"提交"** 或 **"下一步"** 按钮

---

### 第4步：验证网站所有权（重要！）

百度需要确认您是网站的所有者。

#### 方式1：文件验证（推荐）

**4.1 下载验证文件**

百度会提供一个HTML文件，文件名类似：
```
baidu_verify_xxxxx.html
```

**4.2 上传文件到项目**

将下载的文件放到项目的 `public/` 目录：
```
public/baidu_verify_xxxxx.html
```

**4.3 推送到GitHub**

告诉我验证文件的内容，我会帮您添加到项目中。

**4.4 等待部署**

推送后等待1-2分钟，Vercel自动部署

**4.5 点击验证**

回到百度站长平台，点击 **"验证"** 按钮

---

#### 方式2：HTML标签验证

**4.1 获取验证代码**

百度会提供一段HTML代码，类似：
```html
<meta name="baidu-site-verification" content="xxxxxx">
```

**4.2 告诉我验证代码**

复制验证代码告诉我，我会添加到网站中。

---

#### 方式3：CNAME验证（需要自定义域名）

如果您有自己的域名，可以配置CNAME记录验证。

---

### 第5步：提交网站地图

**5.1 找到sitemap提交入口**

验证成功后：
- 进入 **"数据引入" → "链接提交"**
- 或 **"数据监控" → "链接提交"**

**5.2 提交sitemap地址**

输入：
```
https://yuyue8674.vercel.app/sitemap.xml
```

**5.3 提交**

点击 **"提交"** 按钮

---
