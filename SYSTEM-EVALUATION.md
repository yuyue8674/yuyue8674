## 📋 四、立即行动清单

### 本周必做（P0）：

#### 1. 系统上线
```bash
# 1. 构建生产版本
pnpm run build

# 2. 部署到云服务器（阿里云/腾讯云）
# 3. 绑定域名
# 4. 配置HTTPS

# 5. 提交sitemap
curl http://data.zz.baidu.com/urls?site=https://your-domain.com&token=YOUR_TOKEN \
  -H "Content-Type:text/plain" \
  --data-binary @sitemap.xml
```

- 百度地图：https://lbsyun.baidu.com/

  - 登录 → 地点管理 → 新增地点 → 提交审核
  - 预计3-5天审核

- 访问：https://baike.baidu.com/business/
- 使用系统生成的词条内容
