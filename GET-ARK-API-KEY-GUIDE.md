# 🔑 豆包API密钥获取完整流程

---

## 📋 前置准备

- 手机号（用于注册）
- 实名认证（中国大陆身份证）

---

## 🎯 操作步骤

### 第1步：访问火山引擎控制台

在浏览器中打开：
```
https://console.volcengine.com/ark
```

---

### 第2步：注册/登录账号

#### 如果您还没有账号：

**2.1 点击注册**

页面会显示登录/注册选项，点击 **"注册"**

**2.2 填写注册信息**

| 字段 | 填写内容 |
|------|----------|
| 手机号 | 您的手机号 |
| 验证码 | 获取并输入验证码 |
| 密码 | 设置登录密码 |

**2.3 完成注册**

点击 **"注册"** 按钮

---

### 第3步：实名认证

**重要**：使用豆包大模型API需要完成实名认证

**3.1 进入实名认证**

登录后，系统可能会提示需要实名认证

**3.2 选择个人认证**

- 选择 **"个人认证"**
- 填写真实姓名和身份证号

**3.3 完成认证**

按照提示完成人脸识别或其他验证方式

---

### 第4步：开通方舟服务

**4.1 访问方舟控制台**

```
https://console.volcengine.com/ark
```

**4.2 开通服务**

如果是首次访问，点击 **"立即开通"** 或 **"免费试用"**

**4.3 同意协议**

阅读并同意服务协议

---

### 第5步：创建API Key

**5.1 进入API Key管理**

在方舟控制台左侧菜单，找到：
```
API Key管理
```
或直接访问：
```
https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey
```

**5.2 创建新的API Key**

点击 **"创建新的API Key"** 按钮

**5.3 填写信息**

| 字段 | 填写内容 |
|------|----------|
| 名称 | `geo-customer-system` 或任意名称 |
| 描述 | GEO获客系统（可选） |

**5.4 确认创建**

点击 **"确定"** 或 **"创建"**

---

### 第6步：复制API Key

**6.1 查看API Key**

创建成功后，页面会显示API Key

**6.2 复制保存**

⚠️ **重要**：
- API Key只显示一次，请立即复制保存
- 格式示例：`ark-xxxxxxxxxxxxxxxxxxxxxxxx`

**6.3 安全保管**

- 不要分享给他人
- 不要提交到公开代码仓库
- 妥善保管

## ⚠️ 常见问题

### Q1: 注册时收不到验证码？

**解决**：
- 检查手机号是否正确
- 检查短信是否被拦截
- 尝试重新获取

### Q2: 实名认证失败？

**解决**：
- 确保姓名和身份证号匹配
- 确保身份证在有效期内
- 联系客服协助

### Q3: API Key创建后找不到了？

**解决**：
- API Key只显示一次
- 如果丢失，需要重新创建新的Key

### Q4: 没有免费额度？

**解决**：
- 查看是否有活动赠送
- 检查账号是否已完成实名认证
- 联系客服咨询
